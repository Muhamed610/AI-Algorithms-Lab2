# AI-Algorithms-Lab2
СРО №2: Алгоритмы и структуры данных.
Студент: Абдрашитов Мухамед.
Тема: Калькулятор выражений со стеком.
Цель: Освоить работу со структурой данных «стек».
Ответы на контрольные вопросы: 
  Что означает принцип LIFO? 
     Это принцип «Last In, First Out» (последним пришел — первым ушел). 
  Какие операции выполняются со стеком?   
     Основные операции: добавление элемента (push) и удаление/извлечение (pop). 
  Почему стек удобен для обработки выражений? 
     Он позволяет эффективно хранить промежуточные результаты и соблюдать приоритет операций (например, сначала умножение, потом сложение).
