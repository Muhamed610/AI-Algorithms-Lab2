### Ответы на контрольные вопросы:

#### 1. Что означает принцип LIFO? 
Это принцип **«Last In, First Out»** (последним пришел — первым ушел). В этой структуре данных доступен только верхний элемент.

#### 2. Какие операции выполняются со стеком? 
Основные операции включают:
* **push** — добавление элемента на вершину стека;
* **pop** — удаление и извлечение элемента с вершины стека.

#### 3. Почему стек удобен для обработки выражений? 
Стек позволяет эффективно хранить промежуточные результаты и соблюдать приоритет математических операций (например, сначала выполнять умножение, а затем сложение).
